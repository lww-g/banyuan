## 命令行
|命令|说明|
|----|----|
|ls|查询文件|
|mkdir|创建文件夹|
|cd | 进入指定文件夹|
|pwd|显示当前路径|

## 路径说明
* . 当前路径
* .. 上一级路径
* ~ 家路径
* / 起始的是绝对路径
* 非/ 起始的是相对路径，相对于当前路径的路径


## git命令
|命令|说明|
|----|----|
|init|初始化git仓库|
|add|添加工作区文件快照到暂存区|
|commit|添加暂存区的文件内容到本地仓库|
|remote add |添加远程仓库关联|
|push|推送commit到远程分支|
|pull|从远程分支获取文件内容|
|clone|复制远程仓库的内容|
